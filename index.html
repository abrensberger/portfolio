About Me

Web Designer / Developer

My name is Aaron Brensberger and I am a front-end web designer from Brooklyn, New York. I've been designing websites and working with HTML and CSS for 13 years.

I strive to develop clean, reusable code that is both innovative and adaptive, every time I undertake a project.

Learn More

  • Hand code HTML, XHTML, XML and CSS into Ektron, proprietary CMSs, and Drupal
  • Revamp hundreds of websites under the Bootstrap framework
  • Frequently innovate competitive designs which repeatedly result in higher sales
  • Create custom web pages from PDF files
  • Design branding banners, graphics and logos
  • Create A/B split tests for websites using VWO service
  • Design new templates for departmental-wide overhauls
  • Design wireframes and mockups in Photoshop
  • Create, edit, and organize site’s entire file and graphic libraries manually
  • Perform scheduled updates to the site including code and graphic changes
